# Task 3 - Multithreaded Chat Application

This Java project demonstrates a simple chat server and client using multithreading and sockets.

## Files:
- Task3_ChatServer.java: Server code that handles multiple clients.
- Task3_ChatClient.java: Client code to connect and chat with server and others.
- Run multiple clients in different terminals to test chat.

## Instructions:
1. Compile both server and client programs.
2. Run Task3_ChatServer.java.
3. Start multiple instances of Task3_ChatClient.java in different terminals.
4. Start chatting!
