# Task 4 - AI-Based Recommendation System

This Java program uses Apache Mahout to build a user-based recommendation system.

## Requirements:
- Java
- Apache Mahout (version 0.9 recommended)

## Files:
- Task4_UserBasedRecommender.java: Main Java code for the recommendation system
- data.csv: Sample dataset in format (userID,itemID,rating)

## How to Run:
1. Add Apache Mahout to your classpath (via Maven or manually).
2. Place data.csv in the same directory as the Java file.
3. Compile and run Task4_UserBasedRecommender.java

## Output:
The program will print recommended items for user 1 based on user similarity.
